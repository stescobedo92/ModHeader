# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2024-01-13

### Added
- Initial release of ModHeader
- HTTP proxy server with dynamic header manipulation
- Support for adding, modifying, and removing HTTP headers
- REST API for managing header rules
- Request and response header modification
- Rule-based header management with enable/disable capability
- Health check endpoint
- Comprehensive test suite with 93%+ coverage
- Docker support
- Kubernetes deployment configuration
- CI/CD pipeline with GitHub Actions
- Full documentation and examples

### Features
- Dynamic header management via REST API
- Support for both request and response headers
- Toggleable rules without deletion
- Transparent proxying
- Built-in health monitoring
- Cloud-native deployment ready
